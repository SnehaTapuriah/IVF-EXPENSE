Google Sheets Setup (one-time, ~10 minutes)

This app reads & writes directly to your Google Sheet using the official Sheets API. Your data stays in your account — Anthropic never sees it.

Step 1 — Create your Google Sheet

  1. Go to sheets.google.com and create a new blank spreadsheet.
  2. Copy the Spreadsheet ID from the URL: https://docs.google.com/spreadsheets/d/SPREADSHEET_ID/edit

Step 2 — Create a Google Cloud project

  1. Go to console.cloud.google.com and sign in.
  2. Click the project dropdown (top bar) → New Project → name it "IVF Expense" → Create.
  3. Once created, make sure that project is selected in the top bar.

Step 3 — Enable the Sheets API

  1. Go to this link (Sheets API library).
  2. Click Enable.

Step 4 — Configure OAuth consent screen

  1. Go to APIs & Services → OAuth consent screen in the left menu.
  2. Choose External → Create.
  3. App name: "IVF Expense" — User support email: your email — Developer email: your email → Save and Continue.
  4. Skip Scopes (Save and Continue).
  5. On Test users, click + Add Users and add the Gmail addresses of everyone who will use the app (you, your wife, staff). Save.

Step 5 — Create OAuth Client ID

  1. Go to APIs & Services → Credentials.
  2. Click + Create Credentials → OAuth client ID.
  3. Application type: Web application.
  4. Name: "IVF Expense Web".
  5. Under Authorized JavaScript origins, click + Add URI and paste your GitHub Pages URL (e.g. https://your-name.github.io). Use the origin only — no path.
  6. Click Create. Copy the Client ID shown (it ends with .apps.googleusercontent.com).

Step 6 — Create API Key

  1. Still in Credentials, click + Create Credentials → API key.
  2. Copy the key. (Optionally restrict it to Sheets API for security.)

Step 7 — Paste credentials below and connect

Once connected on this device, the Client ID + API Key are remembered locally. You'll only need to sign in with Google once per device (token auto-refreshes).
