Google Sheets Backend Setup

This app uses Google Sheets as the database so all your data syncs across devices.
Follow these one-time steps:

  1. Open Google Sheets and create a new blank spreadsheet.
  2. Click Extensions → Apps Script.
  3. Delete the default code, paste the contents of IVF_AppScript_Backend.gs (provided alongside this file), and click Save.
  4. Click Deploy → New deployment → Web App.
  5. Set "Execute as" = Me and "Who has access" = Anyone.
  6. Click Deploy, authorise the script, then copy the Web App URL.
  7. Paste it below and click Save & Connect.
